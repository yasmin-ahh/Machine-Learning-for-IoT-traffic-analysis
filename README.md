# Machine-Learning-for-IoT-traffic-analysis

IoT devices are computing devices capable of connecting over the network and providing many services and features. However, vulnerabilties and weaknesses in implementation that are constantly discoverd opens the door for inumerous cyber attcks and unauthorized access.

In this project, machine learning techniques and algorithms are used to detect malware and specific stages of DDOS attack in IoT network traffic. 

The dataset used is from the IoT-23 dataset, which is a labeled dataset with malicious and benign IoT network traffic. There are over 23 different scenarios/captures collected from IoT network traffic and different malwares such as Mirai, Torii, IRCBot are executed. 

We used multiple captures, combined many data, cleaned dataset, used Pearson correlation coefficient to improve the quality of the data, reduce the computation time, and improve accuracy.
